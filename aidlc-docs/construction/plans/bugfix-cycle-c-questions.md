# 첨부파일 기능 요구사항 확인 질문 (사이클 C)

현재 상황:
- DB: Message.attachments 필드(JSON)가 이미 존재
- UI: 파일 선택 UI는 있으나 실제 업로드 미구현
- 인프라: AWS S3 SDK가 이미 설치되어 있음

각 `[Answer]:` 태그 뒤에 알파벳을 입력해주세요.
완료되면 "다했어" 또는 "완료"라고 알려주세요.

## Question 1
첨부파일을 어느 시점에 업로드할까요?

A) 티켓 접수 시 — 문의 작성 폼에서 파일 선택 후 접수 버튼 클릭 시 함께 업로드
B) 메시지 작성 시 — 처리자/임직원이 답변 작성 시에도 첨부파일 추가 가능
C) A + B 모두 — 티켓 접수 시와 메시지 작성 시 모두 첨부 가능
D) Other (직접 기술해주세요)

[Answer]: C

## Question 2
첨부파일 다운로드 방식을 선택해주세요.

A) Presigned URL — S3에서 임시 URL 발급 (15분 유효), 클릭 시 직접 다운로드
B) 서버 프록시 — API를 통해 서버에서 S3 파일을 받아 클라이언트에 전달
C) Other (직접 기술해주세요)

[Answer]: A

## Question 3
첨부파일 접근 권한을 선택해주세요.

A) 티켓 접근 권한과 동일 — 해당 티켓을 볼 수 있는 사람은 첨부파일도 다운로드 가능
B) 업로더만 다운로드 가능
C) Other (직접 기술해주세요)

## Question 4

A) 파일당 10MB, 최대 5개, 이미지/PDF/Office 문서/텍스트 허용
